## Setup

```
git clone git@github.com:piisalie/simple_api.git
cd simple_api
./setup
cd playground
```

## Helpful ERL flags

* -noshell
* -pa <directory>
* -boot <file> (sans .boot suffix)
* -compile
* -s <module> <function>
* -extra
